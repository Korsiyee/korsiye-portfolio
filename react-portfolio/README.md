# React Portfolio (converted)

This is a Vite + React scaffold converted from the original static portfolio.

Quick setup:

1. Copy the `assets` folder from the root of your repository into `react-portfolio/public/assets`.
2. From `react-portfolio` run:

```bash
npm install
npm run dev
```

Notes:

- Assets are expected at `/assets/...` inside the Vite `public` folder.
- The app is minimal and focuses on structure + modern styling. You can extend components under `src/components`.
